# Vitrine Digital — Plataforma de Exposição de Produtos e Serviços
Projeto desenvolvido como parte da Residência em Back-End Python.

Este projeto tem como objetivo desenvolver uma plataforma de vitrine digital, onde produtos e serviços podem ser exibidos de forma organizada, permitindo que usuários visualizem informações essenciais de maneira simples e intuitiva.
O foco principal é o desenvolvimento do back-end utilizando Python e Django.
